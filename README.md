# Audio Transcriber 录音转文字应用(本Readme由 AI 生成，内容仅供参考，请仔细甄别。)

功能特性
* 非实时性的录音转文字
* 本地音频文件转文字
* 历史记录管理
* 可配置后端服务地址
* Material Design 3界面设计

技术栈
- Kotlin
- Jetpack Compose
- Room数据库
- OkHttp网络请求
- AndroidX核心组件

构建要求
* Android Studio Giraffe | 2022.3.1+
* JDK 11+
* Android Gradle Plugin 8.0+
* Gradle 8.0+

安装指南
1. 克隆仓库：
git clone https://github.com/bigbirdbrother/audio-transcriber.git
2. 打开项目：
使用Android Studio打开项目根目录
3. 构建应用：
Debug版本：./gradlew assembleDebug   Release版本：./gradlew assembleRelease

项目结构
app/src/main/java/com/bigbirdbrother/recordaudio/
包含以下目录：
res/ - 资源文件
